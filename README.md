# NEUCLID

Neuclid is a neuro-symbolic AI SaaS platform designed to generate high-precision scientific figures (2D Geometry, Physics, etc.) using natural language.

## Architecture

- **Backend:** Python (FastAPI), FireBase, LiteLLM.
- **Frontend:** Angular (v17+).

## Project Structure

- `/backend`: The API and AI Engine logic.
- `/frontend`: The web interface (User Dashboard & Editor).
